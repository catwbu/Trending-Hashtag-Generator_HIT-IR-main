

## 測試文句:

80/20法則，起源於20世紀初由義大利統計學家提出。他指出，在任何特定群體中，
重要因素通常只占少數，而不重要因素則占多數。因此，只要能控制少數重要因素，
就能控制全局。這個原理經過多年演化，成為當今管理學界熟知的二八法則，
即80%的公司利潤來自20%的重要客戶，其餘20%的利潤來自80%的普通客戶。
